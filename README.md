## Before you start

Before you start to use Mantis you need to install:

- [Node.js](https://nodejs.org/)

## Readings

- [React](https://reactjs.org/)
- [Redux](https://redux.js.org/docs/basics/UsageWithReact.html)
- [Sass](http://sass-lang.com/)

## Quick start

Several quick start options are available:

- [Download the latest release](https://github.com/davidedantonio/react-redux-demo/archive/master.zip).
- Clone the repo: `git clone https://github.com/davidedantonio/react-redux-demo.git`.
- Run `npm install`
- After installation run `npm start`
